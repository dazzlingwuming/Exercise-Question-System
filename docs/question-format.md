# 题库格式说明

## 推荐格式

```markdown
### Part 2-015｜单选题｜进阶｜高仿真题

**题目：** 题干内容

**选项/材料：**
- A. 选项 A
- B. 选项 B

**标准答案：** B

**详细解析：** 解析内容

**考察点：** 工程取舍

**常见错误：** 常见误区

**面试延伸追问：** 追问内容

**评分标准：** 答对得 1 分
```

## 当前兼容范围

- 标题可带 `###`，也可不带。
- 字段名可加粗，也可不加粗。
- 支持中文冒号 `：` 和英文冒号 `:`。
- 支持中文竖线 `｜` 分隔标题元信息。
- 选项支持 `- A. xxx` 和 `A. xxx`。
- 多选答案支持 `A、B、D`、`A,B,D`、`ABD`、`a b d`。
- 判断题支持 `正确/错误/对/错/true/false/T/F/是/否`。
- 填空题标准答案可用 `/`、`或`、`、` 分隔多个可接受答案。

## 网页修改与 Markdown 的关系

`Agent 应用层岗位题库.md` 是初始导入来源。题库导入 SQLite 后，刷题、判分、编辑和统计都使用数据库中的运行时版本。

用户在网页中修改题干、选项、标准答案或解析时，系统只更新 SQLite，并写入修改历史；不会自动回写原始 Markdown 文件。这样可以避免自动改写原始题库造成不可控的格式变化。

如果后续需要把修正后的数据库题库重新导出为 Markdown，可以单独增加“导出 Markdown”功能。

## 字段说明

- `题目`：必填，作为题干。
- `选项/材料`：选择题用于选项，主观题可作为材料。
- `标准答案` 或 `参考答案`：必填，用于判分或自评。
- `详细解析`：建议填写，缺失会生成 warning。
- `考察点`：用于筛选和高频错误统计。
- `常见错误`、`面试延伸追问`、`评分标准`：用于提交后的复盘。

## 常见格式错误

- 缺少题干。
- 缺少标准答案或参考答案。
- 选择题选项没有 A/B/C/D 标识。
- 标题没有包含题型。
- 多个字段写在同一行但缺少明确字段名。
